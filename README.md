# About
A simple calculator to help me learn c++

# Prerequisites
g++ Compiler or clang++ Compiler

# Build Instructions

## g++

```cmd
git clone https://github.com/ast-abo/Calculator
cd Calculator
make
./Calculator
```

## Clang++
```cmd
git clone https://github.com/ast-abo/Calculator
cd Calculator
make CXX=clang++
./Calculator
```
